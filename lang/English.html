What is Markdown?

Markdown is a lightweight and readable markup language that easily converts to HTML. It was created by John Gruber and Aaron Swartz in 2004 and is widely used today in documentation, README files, forums, and content management systems.

On this page, you will learn all aspects of Markdown completely with multiple examples.

Headings

Use # to create headings. The number of # determines the heading level.

Markdown:
# Level 1 ## Level 2 ### Level 3 #### Level 4 ##### Level 5 ###### Level 6
HTML Output:
<h1>Level 1</h1> <h2>Level 2</h2> <h3>Level 3</h3> <h4>Level 4</h4> <h5>Level 5</h5> <h6>Level 6</h6>

Text Formatting

Use these symbols to create bold, italic, and strikethrough text.

Markdown:
**Bold Text** __Bold Text__ *Italic* _Italic_ ~~Strikethrough~~
HTML Output:
<strong>Bold Text</strong> <strong>Bold Text</strong> <em>Italic</em> <em>Italic</em> <del>Strikethrough</del>

Lists

Ordered and unordered lists in Markdown.

Markdown:
- List item 1 - List item 2 - Sub item 1 - Sub item 2 - List item 3 1. First 2. Second 3. Third
HTML Output:
<ul> <li>List item 1</li> <li>List item 2 <ul> <li>Sub item 1</li> <li>Sub item 2</li> </ul> </li> <li>List item 3</li> </ul> <ol> <li>First</li> <li>Second</li> <li>Third</li> </ol>

Links & Images

How to add links and images.

Markdown:
[Link Text](https://example.com) ![Image Alt](image.jpg)
HTML Output:
<a href="https://example.com">Link Text</a> <img src="image.jpg" alt="Image Alt">

Code Formatting

Display inline code or code blocks.

Markdown:
Inline: `console.log('Hi')` ```javascript function test() { console.log("Hello"); } ```
HTML Output:
Inline: <code>console.log('Hi')</code> <pre><code class="language-javascript"> function test() { console.log("Hello"); } </code></pre>

Blockquotes

Create a quote block.

Markdown:
> This is a quote. > > - Author
HTML Output:
<blockquote> <p>This is a quote.</p> <p>- Author</p> </blockquote>

Advanced Features

Tables

Create tables using vertical bars:

Markdown:
| Column 1 | Column 2 | Column 3 | |---------|----------|---------| | Cell 1 | Cell 2 | Cell 3 | | Cell 4 | Cell 5 | Cell 6 |
HTML Output:
<table> <thead> <tr>Column 1Column 2Column 3 </thead> <tbody> Cell 1Cell 2Cell 3 Cell 4Cell 5Cell 6 </tbody> </table>

Horizontal Line

Examples:

Markdown:
--- *** ___
HTML Output:
<hr> <hr> <hr>

Reference Links

Another way to create links:

Markdown:
This is a [link][1] and another [link][2]. [1]: https://example.com [2]: https://example.org "Title"
HTML Output:
This is a <a href="https://example.com">link</a> and another <a href="https://example.org" title="Title">link</a>.
